# Feedback

All members and admins can leave feedback for CertGroundâ„¢. This can only be done when you are logged into the application. However no identifying information is stored with your feedback unless you provide your email address; which is optional.

### Types

There are 3 types of feedback that you can provide.

- General Feedback
- Feature Request
- Issues

### Rating

You can leave a rating of 1 to 5.

### Submission

Submissions must contain either the rating and/or the comments section. Otherwise there is no reason to store the feedback.
